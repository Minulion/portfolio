## My Portfolio

A modern and interactive portfolio website built with **Next.js**, **React**, **Tailwind CSS**, **Framer Motion**, and **Three.js** to showcase my work, skills, and projects.

## Stack

- **Next.js**
- **React**
- **Tailwind CSS**
- **Framer Motion**
- **Three.js**
